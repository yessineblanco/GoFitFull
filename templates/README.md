# Email Templates

This directory contains HTML email templates for Supabase authentication emails.

## Files

- `supabase-email-template.html` - Full-featured email template with branding
- `supabase-email-template-simple.html` - Simplified email template

## Usage

These templates can be customized and uploaded to Supabase for custom authentication emails.

See `docs/setup/EMAIL_TEMPLATE_INSTRUCTIONS.md` for detailed instructions on how to use these templates.
