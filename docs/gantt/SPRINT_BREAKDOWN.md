# Sprint Breakdown - GoFit Project

## Sprint Structure
- **Sprint Duration**: 1-2 weeks (mostly 2 weeks)
- **Total Sprints**: 22 sprints
- **Project Duration**: 26 Jan - 26 Jul 2026 (6 months)

---

- Spécifications techniques

### Sprint 1.2 - UI/UX Design & Maquettes (1 semaine)
**Dates**: 2 Fév - 9 Fév 2026

### Sprint 1.3 - Setup Environnement Dev (1 semaine)
**Dates**: 9 Fév - 16 Fév 2026  
**Deliverables**:
- Configuration environnement iOS (Xcode)

## Notes Importantes

1. **Sprints de 1 semaine** : Pour les tâches plus simples ou urgentes
2. **Sprints de 2 semaines** : Pour les fonctionnalités complexes nécessitant plus de développement
3. **Buffer intégré** : Phase 8 (Déploiement) a 4 semaines pour gérer les délais de review des stores
4. **Flexibilité** : Les dates peuvent être ajustées selon les besoins, mais la fin du projet (26 Jul) est fixe
