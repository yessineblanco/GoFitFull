## Instructions pour créer le diagramme

### Option 1 : Excel / Google Sheets
1. Ouvrez Excel ou Google Sheets
2. Importez le fichier `diagramme_gantt.csv`
3. Sélectionnez les colonnes Tâche, Début, Fin
4. Créez un graphique en barres empilées
5. Formatez comme diagramme de Gantt

### Option 2 : GanttProject (gratuit)
1. Téléchargez GanttProject : https://www.ganttproject.biz/
2. Créez un nouveau projet
3. Ajoutez chaque phase avec les dates correspondantes
4. Exportez en PDF ou image

### Option 3 : Microsoft Project
1. Ouvrez Microsoft Project
2. Importez le CSV
3. Le diagramme sera généré automatiquement

### Option 4 : Outils en ligne
- **TeamGantt** : https://www.teamgantt.com/ (gratuit pour petits projets)
- **Smartsheet** : https://www.smartsheet.com/
- **Asana** : avec vue Gantt
