# Uizard Prompt (Short Version - Ready to Paste)

**Redesign the authentication UI for GoFit, a modern fitness tracking mobile app.**

## Brand Colors
- Primary: `#84c441` (vibrant lime green)
- Background: `#FAFBFC` (soft white)
- Text: `#1A1D21` (dark gray)

## Font
Montserrat Alternates (Bold/SemiBold/Regular)

## Screens Needed
1. Welcome Screen - Entry point with "Sign Up" and "Log In" CTAs
2. Login Screen - Email, password, forgot password link
3. Signup Screen - Email, password, confirm password, terms checkbox
4. Forgot Password - Email input, send reset link
5. Reset Password - New password fields
6. Verify OTP - 6-digit code input boxes
7. Password Changed Success - Confirmation screen

## Design Style
- Modern, clean, energetic (fitness-focused)
- Gradient backgrounds with green brand colors
- Card-based forms with subtle shadows
- Rounded input fields (12-16px radius)
- Green primary buttons (`#84c441`)
- Smooth animations and micro-interactions
- Mobile-first (iOS/Android, 375-428px width)

## Key Requirements
- Input fields: Clean borders, floating labels/placeholders, green focus states, error states with red borders
- Buttons: Prominent green primary buttons, loading states, rounded corners (20-24px)
- OTP screen: 6 individual digit boxes, auto-advance, paste support
- Visual feedback: Loading spinners, success animations, error messages
- Accessibility: High contrast, readable text, 44pt+ touch targets

## Inspiration
Strava, Nike Training Club, modern auth UIs like Stripe/Linear - but with energetic fitness branding.

## Deliverables
High-fidelity mockups with specifications (spacing, typography, colors, states) for all 7 screens. Mobile layouts considering safe areas.
